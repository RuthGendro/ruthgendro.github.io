Fishing Map

Built this web app to display a map of lakes that my SO and I have fished at, and how many fish were caught each trip. It's a single-page web application, built using the Knockout framework, that displays a Google Map of an area and all of the lakes we have visited. Users can search all included lakes and, when a marker is selected, additional information about a lake is presented from the FourSquare API. There's also custom map styling and custom marker changes. Trip data appears at the bottom of the info window when it's open.

Technologies Used: JavaScript, jQuery, Knockout, API services, AJAX, version control

Frogger Game

This is HTML5 Canvas powered frogger game where you have a Player and Enemies (Bugs). The goal of the player is to reach the water, without colliding into any one of the enemies. The player can move left, right, up and down. The enemies move in varying speeds on the paved block portion of the scene. Once a the player collides with an enemy, the game is reset and the player moves back to the start square. Once the player reaches the water the game is won.

This game was developed using the best practices in Object Oriented JavaScript, and includes Javscript object.prototype for inheritance, switch statements, for loops, and if/else statements, as well as several classes, which holds various player and enemy functionality.

Technologies Used: HTML5 Canvas, JavaScript, Object Oriented Programming, jQuery, version control
